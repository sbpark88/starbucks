<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

- npm
```shell
npm install npm@latest -g
```

or

```shell
brew install npm
```

### Installation

1. Clone the repo
   ```shell
   git clone https://github.com/sbpark88/starbucks.git
   ```
2. Install NPM packages
   ```shell
   npm install
   ```
3. Run the script
   ```shell
   npm run
   ```
